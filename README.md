# Geode for CLion

Support for [Geode](https://docs.geode-sdk.org/),
the [Geometry Dash](https://store.steampowered.com/app/322170/Geometry_Dash/) modding framework.
Features useful utilities for developing Geometry Dash mods.

Requires the [Geode SDK](https://github.com/geode-sdk/geode)
and [Geode CLI](https://github.com/geode-sdk/cli) to be installed.

This is a plugin similar to the [Geode plugin for VSCode](https://github.com/geode-sdk/vscode),
but for CLion users.

## Features

* Validation for `mod.json`
* Colorization for `FLAlertLayer` tags
* Inline color hints for `ccColor3B`, `ccColor4B`, `ccColor4F` and `ccc3`/`ccc4` macros
* Project template for creating Geode mods in few clicks
* Several lints for common mistakes in mod development
