Reports usage of std::cout in C++ code.

When developing Geode mods, you should use the logging methods from the geode::log namespace instead of std::cout for proper logging integration with the Geode framework.

Example: 


// Bad - not integrated with Geode logging
std::cout << "Hello World" << std::endl;

// Good - use Geode's logging system
geode::log::info("Hello World");
geode::log::warn("Warning message");
geode::log::error("Error message");

The Geode logging system provides better integration with the mod loader and supports different log levels (info, warn, error, debug).
