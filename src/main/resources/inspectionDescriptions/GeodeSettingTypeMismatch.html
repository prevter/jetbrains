Reports C++ type arguments that do not match the setting type declared in mod.json.

The inspection compares the template type used in getSettingValue or setSettingValue with the setting type in mod.json and warns when they are incompatible.

Example: 


// mod.json declares "enable_feature" as type "bool"
auto value = Mod::get()->getSettingValue<int>("enable_feature");

// mod.json declares "accent_color" as type "color"
Mod::get()->setSettingValue<std::string>("accent_color", "red");

Update the C++ type to match the setting type, or update mod.json if the setting type is incorrect.
