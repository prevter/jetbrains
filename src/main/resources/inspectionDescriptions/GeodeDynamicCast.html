Reports usage of dynamic_cast in C++ code.

The dynamic_cast operator is rejected by the Geode mod index and must be replaced with geode::cast::typeinfo_cast. This inspection provides a quick fix to automatically perform the replacement.

Example: 


// Bad - will be rejected
auto* derived = dynamic_cast(basePtr);

// Good - use Geode's type-safe cast
auto* derived = geode::cast::typeinfo_cast(basePtr);

The Geode SDK's typeinfo_cast provides similar functionality to dynamic_cast while being compatible with the mod index requirements.
