Reports Geode setting names that are not defined in the module's mod.json.

This inspection flags calls to getSettingValue or setSettingValue that reference a missing setting, or a setting that is declared as a title entry in mod.json.

Example: 


// mod.json has no setting named "missing_setting"
auto value = Mod::get()->getSettingValue<int>("missing_setting");

// mod.json defines "section_title" as a title entry
Mod::get()->setSettingValue<bool>("section_title", true);

Define the missing setting in mod.json or use a valid, non-title setting name.
